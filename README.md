## About

Across Africa, young people face rising mental-health pressures with little access to affordable, culturally-aware support. At the same time, screen addiction and digital distraction make focus harder than ever.

**NuruMind** ("nuru" means *light*) is a mobile-first companion that helps young people build healthier minds and healthier digital habits — combining mental-wellbeing tools with practical digital discipline.

## ✨ Features

- 🌤️ **Mood check-ins** — track how you feel and spot patterns over time
- 🧘 **Wellbeing tools** — guided exercises for stress, focus, and calm
- 📵 **Digital discipline** — build intentional habits and reduce screen distraction
- 📈 **Personal insights** — understand your progress with simple, encouraging feedback
- 🌍 **Built for African youth** — designed around local context and realities

## 🛠️ Tech Stack

- **Frontend:** React + TypeScript + Vite
- **Styling:** Tailwind CSS
- **Tooling:** ESLint, modern component architecture

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/Gacaca6/NuruMind-App.git
cd NuruMind-App

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Then open the local URL printed in your terminal (usually `http://localhost:5173`).

```bash
npm run build     # Production build
npm run preview   # Preview the production build
```
